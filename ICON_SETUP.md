# 📱 應用程式圖示設定指南

## 需要準備的圖示檔案

為了讓應用程式在 iPhone 上正確顯示圖示，您需要準備以下檔案：

### 必需的圖示檔案

1. **icon-192.png** - 192x192 像素
2. **icon-512.png** - 512x512 像素

### 圖示設計建議

- **尺寸**：必須是正方形（192x192 和 512x512）
- **格式**：PNG 格式
- **背景**：建議使用透明背景或白色背景
- **內容**：可以使用棒球、手套、或應用程式 Logo
- **圓角**：不需要手動加圓角，iOS 會自動處理

### 快速建立圖示的方法

#### 方法 1：使用線上工具
1. 前往 [PWA Asset Generator](https://www.pwabuilder.com/imageGenerator)
2. 上傳您的 Logo 或圖片
3. 自動產生所有尺寸的圖示
4. 下載 `icon-192.png` 和 `icon-512.png`

#### 方法 2：使用設計軟體
1. 使用 Figma、Photoshop、或 Canva
2. 建立 512x512 的設計
3. 匯出為 PNG
4. 使用圖片編輯軟體調整為 192x192

#### 方法 3：使用簡單的圖示
如果暫時沒有設計，可以使用：
- 文字圖示（例如：⚾ 或 "BK"）
- 簡單的幾何圖形
- 棒球相關的 emoji

### 放置圖示檔案

1. 將 `icon-192.png` 和 `icon-512.png` 放到 `public/` 資料夾
2. 確保檔案名稱完全一致（大小寫敏感）
3. 重新部署到 Vercel

### 驗證圖示

部署後，在 iPhone Safari 中：
1. 前往您的網站
2. 點擊「加入主畫面」
3. 檢查圖示是否正確顯示

如果圖示沒有更新：
- 清除 Safari 快取
- 刪除主畫面的應用程式圖示
- 重新加入主畫面

---

## 臨時解決方案

如果暫時沒有圖示檔案，應用程式仍可正常運作，只是會使用預設圖示。您可以稍後再添加自訂圖示。
